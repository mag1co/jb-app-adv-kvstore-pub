1. Install the app

Upload the app ZIP to your YouTrack instance via Admin → Apps.

2. Configure ACL (required)

Go to Administration → Apps → Adv.KVStore → Settings and set allowed groups:

  • Global Scope — Allowed Group — UserGroup with access to global storage
  • Project Scope — Allowed Group — UserGroup with access to project storage

Important: scopes are disabled by default (403). You must set a group to enable access.

3. Open Admin Panel

Go to YouTrack Admin → Adv.KVStore to view stored blocks and access control status.

4. Use from Workflows

Workflows interact with the store via HTTP API.

Endpoints

  • Global: /api/extensionEndpoints/axkvstore/be/{endpoint}
  • Project: /api/admin/projects/{id}/extensionEndpoints/axkvstore/be-project/{endpoint}

Available endpoints: get, set, delete, drop, flush, blocks

See HOW_TO_USE.md for workflow examples, helper scripts, and full API usage guide.
